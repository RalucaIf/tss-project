# Travel Planner Application
A Java-based web application designed to help users organize trips, manage travel itineraries, and upload media from their journeys.

# Features
- Trip Management: Create and organize detailed travel plans.

- Dynamic Itineraries: Pagination for daily activities (e.g., day-1, day-2).

- Media Uploads: Integrated image upload system for capturing travel moments.

- Leaderboard: Track rankings based on user activity.

- Rewards & Points: Earn points for planning trips and interacting with the platform.

- Chatbot Integration: AI-assisted travel queries.

# Tech Stack
- Backend: Java, Spring Boot (Maven)

- Frontend: HTML5, CSS3, JavaScript, React

- Build Tool: Maven

- Database: PostgreSQL
